What is CI/CD?

CI/CD means Continuous Integration and Continuous Delivery/Deployment. It helps developers automate testing and deployment after code changes. Continuous Integration checks code whenever updates are pushed. Continuous Delivery and Deployment help publish the project faster and more reliably. This makes software development easier, quicker, and less error-prone.

My Pipeline

When I push my files to the main branch, GitHub Actions starts automatically. It checks the required files, prepares the site files, and deploys them to GitHub Pages. After that, the live website updates online.
